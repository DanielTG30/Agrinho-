Entendendo as Linguagens da Web

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação fundamental para criar a estrutura e o conteúdo de uma página web. É como se fossem os tijolos e a planta baixa de uma casa, definindo onde cada elemento (textos, imagens, links, etc.) estará localizado.

  • Define a estrutura do documento.
  • Organiza o conteúdo em títulos, parágrafos, listas, etc.
  • Insere imagens e outros elementos multimídia.
  • Cria links entre páginas.

Em resumo, o HTML é a base estrutural do seu site.

O que é CSS?

CSS (Cascading Style Sheets) é a linguagem de estilo utilizada para controlar a apresentação visual de um documento HTML. É como a pintura, os móveis e a decoração de uma casa, definindo cores, fontes, layouts e outros aspectos visuais.

  • Define cores de fundo e texto.
  • Controla o tamanho e o estilo das fontes.
  • Define o layout e o posicionamento dos elementos.
  • Cria designs responsivos para diferentes dispositivos.

Em resumo, o CSS é responsável pela aparência do seu site.

O que é JavaScript?

JavaScript (JS) é uma linguagem de programação que adiciona interatividade e dinamismo às páginas web. É como a parte elétrica e os aparelhos da casa, permitindo que a página responda às ações do usuário e realize tarefas complexas.

  • Adiciona interatividade a elementos HTML.
  • Manipula o conteúdo e o estilo da página dinamicamente.
  • Responde a eventos do usuário (cliques, movimentos do mouse, etc.).
  • Comunica-se com servidores (através de AJAX).
  • Pode ser usado no frontend (navegador) e no backend (Node.js).

Em resumo, o JavaScript adiciona comportamento e interatividade ao seu site.

Em Resumo Geral:

Pense no HTML como o *esqueleto*, no CSS como a *roupa* e no JavaScript como o *comportamento* do seu site. O HTML dá a estrutura e o conteúdo, o CSS cuida da aparência e o JavaScript adiciona a interatividade.
